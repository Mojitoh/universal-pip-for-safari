# Security

Universal PiP for Safari runs as a Safari WebExtension content script. It does not collect analytics, phone home, or store browsing data.

## Permissions

The extension requests broad page access so it can detect HTML5 videos across sites and accessible frames. Safari still requires the user to approve and enable the extension.

## Reporting a vulnerability

Please open a private security advisory on GitHub if available, or create an issue with enough detail to reproduce the problem.
